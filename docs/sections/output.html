Output

The code generates a print output containing details of the calculations and results and a HDF5 file

Printed output

Input files and parameters summary

The first section of the printed output contains a summary of all the input parameters and input files used to run the code.

Spin hamiltonian

This section prints the unperturbed spin hamiltonian and teh corresponding eigenvalues after diagonalization of the spin hamiltonian.

Initialize simulation

The part contains information related to the parallerization of the calculation.

Eigenvalues of the R matrix

This section prints the eigenvalues of the corresponding Redfield matrix. Note that for a simulation to have physically meaninful results, the Redfield matrix eigenvalues should be one zero which represents the steady state and the rest of the eigenvalues should be negative.

Initial spin population

This section prints the initial spin density.

Time evolution

After performing the time evolution of the spin density, the code computes the magnetization of the system and estimate the T1 decay in the direction set by the initial parameters. Additionally, the code estimates the accuracy of the T1 estimation by computing the R2 and the absolute error.

Run time

The last part of the printed output contains information related to the time required to run each part of the code.
